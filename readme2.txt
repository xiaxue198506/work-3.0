this doc tell you how to set up an local respository and push it to remote respository.
1. set ssh key between local and remote.  
   First set a local one and a remote one. 
   Second init a ssh key in local if you don't have one.
   Third add ssh key in remote. And commit your local repository files.
   Fourth use git remote add origin git@github.com:dfdfdsdfdfd.git ,put local to remote named origin.
   Fifth use git push -u origin master
